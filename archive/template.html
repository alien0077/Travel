📋 旅遊行程 JSON 產生範本

將此頁內容作為參考，在 trips/ 目錄下建立新的 {id}.json 檔案。

1 確認基本資料 — 行程名稱、日期、類型（city/drive）、涵蓋縣市與機場
2 逐日填入行程 — 每天的時間軸、美食、住宿、停車場
3 補充地圖連結 — 每個景點/餐廳/飯店附上 mapUrl，自駕再加 mapcode
4 加入 manifest — 將新的 id 加入 trips/manifest.json

使用流程

  1. 複製上方 JSON 結構到 trips/{id}.json
  2. 逐項填入實際行程資料，刪除不需要的欄位（如 city 模式可刪 parking）
  3. 所有 mapUrl 盡量填寫 Google Maps 短連結
  4. 自駕行程務必填寫 mapcode
  5. 將 id 加入 trips/manifest.json
  6. 開啟 index.html 確認卡片正確出現
  7. 開啟 viewer.html?trip={id} 確認行程內容正確
